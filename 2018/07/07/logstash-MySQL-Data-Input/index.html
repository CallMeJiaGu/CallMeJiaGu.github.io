logstash需要安装logstash-input-jdbc插件，才可以对数据库进行操作。下载插件过程中最大的坑是下载插件相关的依赖的时候下不动，因为国内网络的原因，访问不到亚马逊的服务器。解决办法，改成国内的ruby仓库镜像。此镜像托管于淘宝的阿里云服务器上.。
